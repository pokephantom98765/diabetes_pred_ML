# Diabetes Prediction using Machine Learning

A Machine Learning project that predicts whether a person is diabetic or not based on medical attributes.

---

## Overview

This project uses medical datasets and machine learning algorithms to build a predictive system for diabetes detection. The model analyzes various health-related parameters and predicts the possibility of diabetes.

---

## Features

- Medical data analysis
- Data preprocessing
- Predictive ML system
- Model training and evaluation
- Accuracy testing

---

## Technologies Used

- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab

---

## Dataset Features

- Pregnancies
- Glucose Level
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age

---

## Machine Learning Workflow

1. Data Collection
2. Data Preprocessing
3. Data Visualization
4. Train-Test Split
5. Model Training
6. Prediction System
7. Accuracy Evaluation

---

## Model Used

- Support Vector Machine (SVM)

---

## Results

The model predicts diabetes risk effectively with good prediction accuracy.
